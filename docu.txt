Para obtener ayuda de la lista de comandos
*tsc --help

Para transpilar codigo ts y que se cree el archivo js a partir del ts ya que los navegadores no entienden codigo ts, se utiliza el comando:
*tsc nombre del archivo

Para ejecutar el archio
*node nombre del archivo 

Si queremos que el archivo ts se transpile en otro directorio y se ejecute podemos utilizar el comando combinado
*tsc --outDir dist index.ts &&  node dist/index.js
este comando crea una carpeta llamada "dist" luego transpila el codigo ts a js y por ultimo ejecuta dicho archivo

Para colocar un observador y que todos los cambios del archivo ts se guarden automaticamente en el js
*tsc --outDir dist --watch index.ts
--outDir dist especifica que los archivos JavaScript generados se guardar√°n en el directorio dist.
--watch le dice a TypeScript que observe los cambios en los archivos.
index.ts es el nombre de tu archivo TypeScript principal.

Para crear el archivo "tsconfig.json" se utiliza el comando
*tsc --init
PROPIEDADES
-En la seccion "Emmet" se puede encontrar el "outDir" comentado, lo descomentamos y marcamos la ruta donde vamos a dejar los archivos transpilados, es este caso seria "./dist"
-Otra opcion en la seccion "modules" vamos a encontrar "rootDir", para indicar desde que carpeta vamos a transpilar los ficheros, en este caso es "./src"
-La propiedad "target" nos permite cambiar la version js a la que se transpilan los archivos
-Una vez que hayamos configurado esas propiedades, podemos utilizar el comando "tsc" y ya va a transpilar el codigo de ts a js en la ubicacion designada
